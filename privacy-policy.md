# Política de Privacidad – CARELOGIX Contador DGII 606/607

Esta herramienta es utilizada por CARELOGIX CRG SRL para ayudar a registrar y organizar información de facturas con fines contables y de cumplimiento fiscal.

## Datos que se procesan

- Imágenes de facturas (fotos o PDF) enviadas por el usuario.
- Datos extraídos de las facturas: RNC, NCF, fechas, montos, ITBIS y otros campos necesarios para los reportes 606/607.
- Información técnica necesaria para guardar los datos en servicios de Microsoft 365 (SharePoint, OneDrive, Excel) u otros sistemas configurados por el propietario.

## Uso de la información

- Los datos se utilizan exclusivamente para:
  - Registrar las facturas en una base de datos contable.
  - Generar archivos de reporte para la DGII (formularios 606 y 607).
  - Verificar duplicados y consistencia de la información.

CARELOGIX CRG SRL no vende ni comparte esta información con terceros ajenos al proceso contable, salvo obligación legal.

## Almacenamiento

- Los datos se almacenan en servicios en la nube controlados por el propietario (por ejemplo, Microsoft 365 / SharePoint / OneDrive).
- El acceso a estos datos está limitado a las cuentas autorizadas por el propietario.

## Responsabilidad del usuario

El usuario es responsable de:
- Asegurarse de que las facturas enviadas son legales y correctas.
- No subir información sensible que no sea necesaria para fines contables.
